## 介绍

在这张地图中，两个定居点为了生存正在同一片领地上交战。从当地动物身上获取的资源既可以卖给商人，也可以用来喂养劫掠兽。不过劫掠兽更钟爱用营火烤制的肉——每个定居点的中心地带都燃烧着永恒之火，正是这团火焰维系着营火的长明不熄。

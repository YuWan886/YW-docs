## 介绍

《派出小丑》是在色彩缤纷的马戏团中对捉迷藏游戏的独特改编！

小丑们可以找到增益道具来帮助他们躲藏并为最后一分钟做准备！

在最后一分钟，所有小丑都会发光，而所有警察都会获得速度提升！

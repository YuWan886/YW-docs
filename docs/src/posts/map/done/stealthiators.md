## 介绍

在《寻踪追迹》中偷偷冲向终点线吧！游戏分为两支队伍：跑者与角斗士。角斗士的目标是在跑者抵达终点前击中他们。跑者全程隐形，仅靴子可见，这意味着他们可以躲藏在任何高度超过一个方块的障碍物后面。

本地图包含：
- 随机生成的地图每局游戏
- 八个独特职业角色
- 配备专属配音角色的超大等候大厅
- 三叉戟小游戏

尽情享受潜行乐趣吧！

## 介绍

《偶像对决》是对KitPVP体验的全新诠释，拥有多样化的可玩角色阵容，每个角色都有自己独特的技能需要学习和掌握。从基于Minecraft生物的角色阵容中选择，加入这场对决吧！

## 特色

- 15个可玩角色套装
- 5张地图
- 3种游戏模式（淘金热、团队死亡竞赛、自由混战）
- 可自定义设置

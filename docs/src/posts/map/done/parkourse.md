## 介绍

建造一个跑酷赛道并挑战你的朋友。如果你自己或别人的跑酷失败，就会失去一条命。使用自定义方块来每次创建一个独特的赛道。

查看我们的Discord服务器获取更多自定义地图和小游戏！https://discord.gg/cVPJnDBMEy

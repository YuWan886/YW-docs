# Aghanim's Labyrinth 2 | 阿哈利姆的迷宫 2
![我是封面](https://ypy-uss.yuwan886.top/YW-docs/map/aghanim-labyrinth/cover.png)

- 作者：[**Mr_Kheese**](https://www.planetminecraft.com/member/mr_kheese/)
- 原址：[PMC](https://www.planetminecraft.com/project/aghanim-s-labyrinth-2-roguelike-pve-map/)

- 汉化：[下崽](https://pan.quark.cn/s/17dd5184e1b5)

## 介绍

这是我备受期待的PVE地图续作！原为Dota 2中的自定义模式，《阿哈利姆的迷宫2：连续体谜题》现已在Minecraft中忠实重现。

### 特色
- **肉鸽式怪物竞技场**：通过16层独特的战斗遭遇和深入的Boss战
- **丰富的自定义机制**：大量自定义怪物和物品机制
- **程序生成房间**：随机生成的房间选择增加可重玩性
- **可解锁难度等级**：逐步解锁更高难度挑战
- **自定义法术系统**：玩家可使用并升级自定义法术
- **多平台支持**：支持单人或多人游戏（最多 **7** 人）

### 制作信息
- **开发时间**：超过600小时
- **命令方块**：超过10,000个
- **特别感谢**：
  - 所有帮助测试地图的朋友们
  - Valve Software创造了原版阿哈利姆的迷宫游戏模式
  - Rich Sommer为阿哈利姆配音的精彩表演

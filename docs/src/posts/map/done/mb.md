## 介绍

《重锤爆破手》是一款两支队伍——火花队与天劫队——为胜利而战的 PvP 小游戏。超越你的对手，将他们击落以赢得胜利！

### 🧩 特色

- 垂直战斗：基于动量和策略的立体对战
- 灵活组队：支持 1v1、2v2、3v3 及任意人数组合
- 多语言支持：英语、西班牙语还有简体中文
- 游戏内信息手册：提供完整规则说明
- 永久性竞技场词条：独特永久效果
- 自定义观战模式：专为观战设计

### 💨 比赛规则

- **装备**：下界合金盔甲（保护 IV）
- **生命**：仅有一条生命
- **武器**：重锤 + 无限风弹
- **胜利条件**：击败对方队伍所有成员
- **危险区域**：竞技场边缘有熔岩坑

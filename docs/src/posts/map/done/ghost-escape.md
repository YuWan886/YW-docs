## 介绍

你发现自己身处未知地域，不知道会遭遇什么。你能设法脱身吗？  
地图上有上锁的门，可以用钥匙打开。  
门后面可能有控制面板、发电机、各种物品，也可能什么都没有。  
如果你找到了发电机，可以用扳手和装满汽油的油桶修复。  
如果你找到了控制面板，可以用钳子撬开它，但不要急着放下杠杆，需要先修复发电机。   
该模式适用于1至4人游戏。

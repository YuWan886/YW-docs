## Show Me Log

在游戏错误弹窗中增加打开日志目录和导出日志的按钮，可以使用 Ctrl+L 快捷导出日志，方便快速定位问题。
在设置中可以自定义快捷导出日志的按键，需要配合Ctrl使用。

## Relic Fold

当遗物太多时，可切换为单行滚动显示，避免占满屏幕。按 Ctrl+R 切换折叠/展开。

## Faster Abandonment

放弃当前 run 时跳过结算画面，直接返回主菜单。
只在玩家主动放弃时生效，正常的胜利/失败结算画面不受影响。

- [**更快的放弃-FasterAbandonment**](https://www.nexusmods.com/slaythespire2/mods/1284)


## BucketofAxolotl Skin

将我的世界中的美西螈桶加入尖塔
推荐安装 Ritsulib 可以在 Mod设置(Ritsulib) 中设置替换哪个角色的皮肤
默认替换战士的

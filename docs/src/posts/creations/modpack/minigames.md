# 迷你游戏合集

> 一些有意思的模组:)

## Peak Craft

通过替换原有的健康和饥饿系统，引入全新的耐力系统，你将能够在你的 Minecraft 世界中像在 PEAK 中一样奔跑和攀爬表面。  
要在表面上攀爬，你必须按住右键并向前移动在任何平坦表面上。你也可以在攀爬时按住潜行键来获得冲刺加成，但会消耗一部分耐力。  
你可以通过右键按住一个攀爬玩家，来帮助后面的攀爬玩家。  
如果你不吃东西，你的耐力会以不同的方式减少，开始时会出现饥饿，你的条形图会下降。受到伤害也会减少耐力，每种伤害都会引起特定的症状。其中一些会自行消退，而另一些则需要特殊物品来治疗。  
如果你的疲劳度超过耐力条，你会失去意识，如果未被救醒，你将死亡。  
潜行拾取物品。

## 我的世界2D版

有时候生活在三维空间中可能会让人感到压力，那么为什么不卸下负担，选择生活在二维空间中呢？  
Two Dimensional: Reloaded 通过移除对三维的访问，并将玩家的移动限制在沿 X 轴的单个平面上，将 Minecraft 转化为一个 2D 横版卷轴游戏。  
如果你开始怀念你的三维生活，只需按下 B 键即可访问背景层。  
按住 B 键转身，让你能够在背景层放置和破坏方块。（按键绑定可在 Two Dimensional: Reloaded → Face Away 下配置）。
如果你觉得相机视口大小太小，增加你的 FOV 以看得更远。  
在背景层建造一个下界传送门，右键点击传送门方块即可前往下界（使用原版冷却时间和时间限制）。  
可选方块放置指南功能（使用 Cloth Config API + ModMenu 进行配置）
